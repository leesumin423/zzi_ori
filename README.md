# 주가현황 대시보드

네이버 금융 공개 페이지/API에서 지수·테마·계열사 주가·환율 데이터를 긁어와 한 화면(다크모드)으로 보여주는 Flask 기반 대시보드입니다. 회사명을 클릭하면 최근 5일 수급(기관/외국인) 동향 팝업이 뜨고, Export PDF로 인쇄용 리포트를 바로 뽑을 수 있습니다.

> **개발 지식이 없는 분은** 이 저장소를 통째로 받을 필요 없이 [dist/주가현황_배포.zip](dist/주가현황_배포.zip) 파일 하나만 받아서 압축을 풀고 `주가현황_실행하기.bat`만 더블클릭하면 됩니다. (사용법은 압축 안의 `사용법.txt` 참고)

## 주요 기능

- **지수 현황**: KOSPI/KOSDAQ 현재가 + 투자자 동향 요약
- **테마별 시세**: 시멘트/레미콘, 건설대표주, 증권, 미디어 등 등락률
- **상장 계열사 / 시멘트사 주가 현황**: 주식수·자본금·전년말 대비 시가총액·52주 최고/최저
- **현재기준 / 장마감기준 탭**: 실시간가 vs 15:30 확정 종가 기준을 전환해서 볼 수 있음
- **수급 동향 모달**: 표의 회사명을 클릭하면 최근 5일 개인(추정)/기관/외국인 순매매, 총거래대금 팝업
- **주가 동향 코멘트**: 최근 10영업일 수급 패턴(연속 순매수/매도, 급등락 등)을 분석하고 관련 뉴스 헤드라인을 매칭해 보고서 톤 코멘트를 자동 생성 (표 아래, 인쇄 시 여백을 채움)
- **Export PDF**: 브라우저 인쇄 기능으로 화면 그대로 PDF/인쇄 저장

> 사이드바 "공시" 메뉴(단판공시/지분공시)는 별도로 개발 중인 기능이라 설명을 따로 뺐습니다 — [단판공시_모니터링.md](단판공시_모니터링.md), [지분공시_모니터링.md](지분공시_모니터링.md) 참고.

## 실행 방법

### 요구 사항
- Python 3.9 이상

### 설치
```bash
pip install -r requirements.txt
```

### 실행
```bash
python server.py
```
브라우저에서 `http://localhost:5000` 접속. (Windows에서는 `주가현황_실행하기.bat`를 더블클릭하면 서버 실행 + 브라우저 열기까지 자동으로 됩니다.)

### DART 연동 — 거래정지 사유 · 자본금 자동 확인
DART(전자공시) API로 두 가지를 자동화합니다:
- "주가 동향 코멘트"에서 거래정지가 감지되면 주식병합/소각 결정 공시 원문을 읽어
  정확한 매매거래정지기간까지 알려줍니다.
- 계열사/시멘트사 표의 **자본금**을 재무제표에서 실측값으로 가져옵니다 (예전엔 종목별
  액면가 차이를 반영 못 한 수동 계산값이라 부정확한 경우가 있었습니다).

이 저장소엔 팀 공용 DART API 키(`.dart_api_key`)가 이미 포함되어 있어 **별도 설정 없이
바로 동작**합니다. 다만 이 키는 하루 요청 한도를 여러 명이 같이 나눠 쓰는 것이라, 한도
초과로 안 될 경우 [opendart.fss.or.kr](https://opendart.fss.or.kr)에서 본인 명의로
무료 키를 발급받아(가입 즉시, 승인 대기 없음) `.dart_api_key` 파일 내용을 자기 키로
바꾸거나 환경변수 `DART_API_KEY`로 설정하면 됩니다. 키가 아예 없거나 조회에 실패해도
나머지 기능엔 영향 없습니다(자본금은 하드코딩된 폴백 값으로 표시됨).

⚠️ **이 키는 저장소 안에 평문으로 들어있습니다.** Private 저장소에 초대된 사람,
`dist/주가현황_배포.zip`을 받은 사람 누구나 값을 볼 수 있으니, 외부에 공개로 전환하거나
불특정 다수에게 배포하지 마세요.

## 커스터마이징

전부 [server.py](server.py)의 `/data` 라우트(`data_endpoint` 함수) 안에서 관리합니다.

| 바꾸고 싶은 것 | 위치 |
|---|---|
| 계열사 종목/종목코드 추가·변경 | `data_endpoint()` 안 `COMPANIES` 딕셔너리 |
| 계열사 자본금(억원) | DART API로 자동 조회됨(`fetch_dart_capital`). 새 종목 추가 시 `DART_CORP_CODES`에 고유번호 추가 필요 — DART 키가 없을 때만 같은 곳 `CAPITAL_FALLBACK` 딕셔너리 사용 |
| 시멘트사 종목 | `CEMENT` 딕셔너리 (자본금은 계열사와 동일하게 DART 자동 조회 + `CAPITAL_FALLBACK`) |
| 테마 그룹(및 순서) | `THEMES` 리스트 — 네이버 테마 그룹 번호(`no=`)는 `finance.naver.com/sise/sise_group_detail.naver?type=theme&no=N` URL에서 확인 |
| 주가 동향 코멘트 대상 종목 | `TARGETS` 리스트 (`section == 'commentary'`) |
| 거래정지 사유(DART 미확인 시 대체 문구) | `MANUAL_HALT_REASONS` 딕셔너리 |
| DART 조회 대상 종목의 고유번호 | `DART_CORP_CODES` 딕셔너리 (종목코드와 다른 DART 전용 번호) |

화면 쪽(색상, 레이아웃, 표 구조)은 [styles.css](styles.css) / [주가현황.html](주가현황.html) / [dashboard.js](dashboard.js)에서 직접 수정하면 됩니다.

## 데이터 출처 및 주의사항

화면의 각 숫자가 정확히 어느 URL/API에서 어떻게 오는지는 [데이터_출처.md](데이터_출처.md)에 항목별 표로 정리되어 있습니다.

- 모든 데이터는 네이버 금융(finance.naver.com)의 **비공식 공개 API/페이지를 스크래핑**해서 가져옵니다. 네이버가 페이지 구조나 API를 바꾸면 특정 항목이 갑자기 안 나올 수 있습니다 — 이 경우 서버 콘솔에 `[DEBUG]` 로그가 출력되니 어디서 실패했는지 확인할 수 있습니다.
- 개인/사내용 참고자료 용도로 만들어졌습니다. 짧은 주기로 자동 새로고침하거나 대량 배포하는 용도로는 사용하지 마세요 (네이버 서버에 부담을 줄 수 있고 서비스 약관에 어긋날 수 있습니다).
- "주가 동향 코멘트"의 개인 순매매·매수/매도 원인 설명은 공개 데이터로 추정한 근사치입니다(기관·외국인 순매매의 잔차로 개인 수급을 추정, 같은 날짜에 종목명이 포함된 뉴스 헤드라인을 매칭). 투자 판단의 근거로 쓰지 말고 참고용으로만 사용하세요.

## 알려진 제한사항

- 52주 최고/최저, 전년말 주가는 네이버가 계산에 사용하는 정확한 거래일 기준과 소폭 다를 수 있습니다.
- 환율은 USD/KRW만 지원합니다.
- 개인 순매매는 실제 데이터가 아니라 추정치입니다 (위 주의사항 참고).
